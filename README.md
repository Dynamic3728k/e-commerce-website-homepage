# E-Commerce Landing Page Template 

A responsive frontend clone of a modern e-commerce website homepage. This project demonstrates proficiency in **HTML5**, **CSS3**, and **Vanilla JavaScript**, focusing on layout structure, responsive design, and interactive DOM manipulation.

## Important Note on Assets
**Trademark & Copyright Disclaimer:** This project is strictly for educational purposes. To comply with trademark policies and copyright laws:
* **No official brand logos are used.** The logo has been replaced with a generic text placeholder.
* **No proprietary product images are used.** All product and hero images have been replaced with generic placeholders (via `placehold.co`) to prevent copyright infringement and hotlinking issues.

##  Features
* **Responsive Navbar:** Includes search bar, location placeholder, and cart icon.
* **Hero Image Carousel:** Automatic sliding banner with manual previous/next controls.
* **Sidebar Navigation:** Off-canvas sidebar menu with smooth transition effects.
* **Product Grids:** CSS Grid layouts for product showcases (Quads and Single items).
* **Horizontal Scroll:** "Today's Deals" section with horizontal overflow scrolling.
* **Footer:** A complete, multi-column footer layout.

##  Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Flexbox, Grid, CSS Variables (Custom Properties), and Media Queries for responsiveness.
* **JavaScript:** DOM manipulation for the sidebar toggle and image slider logic.
